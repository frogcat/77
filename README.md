# 77: 7x7 のカラム地図 (TabularMaps)

# 使い方
次の URL で 7 x 7 のカラム地図を作ることができます。

```
https://tabularmaps.github.io/77/?title={題名}&prefs={}&color={}
```

- title: カラム地図の左上に書き込むタイトル
- prefs: 色を塗る都道府県のコード
- color: 都道府県に塗る色
